# AchieveChain 🚀

Welcome to **AchieveChain**! 🌟 A blockchain-based personal achievements tracker that allows you to record, manage, and showcase your milestones and accomplishments securely on the Ethereum blockchain. 

## 📜 Overview

**AchieveChain** leverages the power of blockchain technology to give you a transparent and immutable record of your personal achievements. Whether you’re reaching career goals, personal milestones, or just want to keep track of your progress, AchieveChain ensures your achievements are securely logged and easily accessible.

## 🚀 Features

- **Immutable Record**: Store your achievements on the Ethereum blockchain for permanent, tamper-proof records.
- **Easy Interaction**: User-friendly web interface for adding and viewing your achievements.
- **Secure**: Only you can add or view your achievements using your Ethereum address.

## 🛠️ Technologies Used

- **Solidity**: For writing smart contracts.
- **Ethereum**: Blockchain platform for deploying smart contracts.
- **Web3.js**: JavaScript library for interacting with Ethereum.
- **HTML/CSS/JavaScript**: Front-end technologies for the web interface.
- **Truffle**: Development framework for Ethereum smart contracts.

## 📦 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (>=12.x)
- [Truffle Suite](https://www.trufflesuite.com/truffle) (for smart contract development)
- [Ganache](https://www.trufflesuite.com/ganache) (for local Ethereum blockchain)
- [MetaMask](https://metamask.io/) (browser extension for managing Ethereum accounts)

### Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/kathleenmarkowitz/achievechain.git
    cd achievechain
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Start Ganache**:
    Launch Ganache and ensure it’s running on port 8545.

4. **Compile and deploy contracts**:
    ```bash
    truffle compile
    truffle migrate
    ```

5. **Serve the front-end**:
    ```bash
    npx http-server src
    ```

6. **Open the application**:
    Navigate to `http://localhost:8080` in your browser.

## 🎉 Usage

1. **Connect MetaMask**: Ensure MetaMask is connected to the same network as your Ganache instance.
2. **Add Achievement**: Use the web interface to add new achievements.
3. **View Achievements**: Check your list of achievements and view details.
