    ```bash

    cd achievechain
    ```

2. **Install dependencies**:
    ```bash
    npm install
    ```

3. **Start Ganache**:
    Launch Ganache and ensure itâ€™s running on port 8545.

4. **Compile and deploy contracts**:
    ```bash

    ```

5. **Serve the front-end**:
    ```bash
    npx http-server src
    ```

6. **Open the application**:
